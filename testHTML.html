Linking CSS Styles to HTML file
  1. Add link and href to the title of the HTML file to link it to the CSS file
  2. Then type the "type": type="text/css" (for CSS files)
  3. Lastly add the relationship: rel="stylesheet" (for CSS files)
Benefits of Using a separate CSS file (than Internal CSS)
  • "Allows all pages to use the same style rules"
  • "Keeps the content separate from how the page looks"
  • "Means you can change the styles used across all pages by altering just one file (rather than each individual page)
